Introducción Última actualización: 28-10-2024

Bienvenido a la documentación de la API de Chelenko, un conjunto de rutas diseñadas para interactuar de manera eficiente con la base de datos de gestión de huéspedes, habitaciones, reservas, modificaciones de tarifas e inventario del sistema Chelenko. Esta API está destinada a desarrolladores que buscan integrar y gestionar información hotelera mediante operaciones CRUD (crear, leer, actualizar, eliminar) a través de solicitudes HTTP.

Huéspedes

La API de huéspedes permite gestionar la información de los huéspedes mediante diversas rutas. A continuación, se describen las rutas disponibles, sus métodos HTTP y cómo utilizarlas.

GET ALL

GET

http://chelenko-data.sa-east-1.elasticbeanstalk.com/api/guests

Recupera una lista de todos los huéspedes registrados en la base de datos.

Código 200: Devuelve un array con los objetos de los huéspedes.
Código 500: Error en la recuperación de datos.

GET by ID

GET

http://chelenko-data.sa-east-1.elasticbeanstalk.com/api/guests/:id

Recupera la información de un huésped específico mediante su ID.

Parámetros:

ID del huésped a recuperar.

Código 200: Devuelve el objeto del huésped.
Código 404: Si no se encuentra el huésped.

CREATE

POST

http://chelenko-data.sa-east-1.elasticbeanstalk.com/api/guests

Crea un nuevo registro de huésped en la base de datos.

El cuerpo de la solicitud es JSON

[
	{
	"firstName": "Nombre",
	"lastName": "Apellido",
	"email": "correo@example.com",
	"phone": "123456789",
	"nationality": "Nacionalidad",
	"documentType": "PASSPORT",
	"documentNumber": "Número de documento",
	"address": {
		"street": "Calle",
		"city": "Ciudad",
		"state": "Estado",
		"country": "País",
		"postalCode": "Código postal"
			},
	"preferences": { "key": "valor" },
	"blacklisted": false,
	"blacklistReason": "Razón"
						}
						  ]

Código 201: Devuelve el objeto del huésped creado.
Código 400: Si faltan datos requeridos (firstName, lastName, email).

PUT

PUT

http://chelenko-data.sa-east-1.elasticbeanstalk.com/api/guests/:id

Actualiza la información de un huésped existente mediante su ID

Parámetro: ID del huésped a actualizar

El cuerpo de la solicitud es JSON

Se pueden enviar los mismos campos que para crear un nuevo huésped, omitiendo aquellos que no se deseen actualizar.

Código 200: Devuelve el objeto del huésped actualizado.
Código 404: Si no se encuentra el huésped.

DELETE

DELETE

http://chelenko-data.sa-east-1.elasticbeanstalk.com/api/guests/:id

Elimina un registro de huésped específico de la base de datos

Parámetro: ID del huésped a eliminar

El cuerpo de la solicitud es JSON

Código 204: El huésped se ha eliminado correctamente.
Código 404: Si no se encuentra el huésped.
